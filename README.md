# Vibify 🎧
A modern minimal music streaming UI built with React.js.

## Features ✨

- Modern pink aesthetic UI
- Responsive mobile navigation
- Music streaming inspired interface
- Interactive bottom navigation
- Search functionality UI
- Minimal glassmorphism design
- React Router navigation
- Smooth hover interactions

- ## Tech Stack 🚀

- React.js
- Tailwind CSS
- React Router DOM
- Remix Icons

- ## Installation

```bash
git clone https://github.com/diyameghani26/Vibify.git

cd Vibify

npm install

npm run dev
